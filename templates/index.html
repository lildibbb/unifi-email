NEXTGen VIP Email Blast User Guide

Setup instructions, user workflow, and extension installation

Overview

The NextGen VIP Email Blast is a custom email generation and automation portal designed for Telekom Malaysia's VIP onboarding operations. It provides a real-time responsive HTML email preview tool (supporting both English and Bahasa Melayu layout templates) and uses a Chrome Extension to safely inject and automate sending emails directly through the official Microsoft Outlook Web client.

Chrome Extension Installation Guide

To automate email sending directly from this dashboard, you must install the helper Chrome extension included in the codebase.

1 Load Unpacked (Developer Mode)

  1. Locate the extension folder inside this project's directory.
  2. Open Google Chrome and navigate to chrome://extensions/.
  3. Enable the Developer mode toggle switch at the top-right corner.
  4. Click the Load unpacked button at the top-left corner.
  5. Select the extension folder to load it.

Usage Workflow Guide

Follow these steps to generate and dispatch your onboarding email templates:

1

Verify Connection Status Extension missing Connected

Look at the top-right corner status badge. If it shows **Extension missing**, verify the extension is loaded in your browser. When correctly loaded, it will show **Connected** or **Disconnected** state.

2

Authenticate and Bind Outlook Web

Click the Connect Outlook button. This opens a browser tab navigating to Microsoft Outlook Web. The extension will automatically bind this page to establish a secure background communication bridge.

3

Fill Form Details & Select Language

Input the recipient email and customize all customer information, Unifi connection details, and NEXTGen crew details on the left panel. Use the **Language Selection** card to switch the email between English (ENG) and Bahasa Melayu (BM).

4

Verify Live Responsive Preview Desktop Mobile

Verify columns, values, and design system formatting in real time in the right panel. Toggle **Desktop** and **Mobile** views to ensure correct height matching of the key-value sections under both layouts.

Initiate Email Send

Click the **Send Email via Outlook** button. The extension will automatically focus on your Outlook tab, open the compose window, populate the recipient and subject fields, inject the custom HTML onboarding email, and send it.
